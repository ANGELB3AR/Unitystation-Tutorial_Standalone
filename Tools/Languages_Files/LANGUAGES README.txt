Folder "languages" need to be placed in LocalLow\Unitystation\unitystation

To make your own translation :

In Files :
	- Copy-Paste one of the existing translation
	- Replace the language tag name (exemple : <English> and </English>) by your language name
	- Replace each sentences by your translation

In Unity Project :
	- Duplicate one of the existing language button in the title screen (search "lang" to find faster)
	- Replace the language name in the text and in the button event script
